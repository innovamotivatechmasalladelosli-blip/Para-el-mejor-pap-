# 🎉 Feliz Día al Mejor Padre

Una página web hermosa y animada para celebrar el Día del Padre, con dedicatorias personalizadas y animaciones interactivas.

## 📋 Características

✨ **Diseño Responsivo**: Funciona perfectamente en dispositivos móviles y de escritorio

🎨 **Animaciones Suaves**: Transiciones y efectos visuales atractivos

💌 **Dedicatorias Personalizadas**: Dos secciones con dedicatorias de Uriel y Vitoria

🎈 **Globo Interactivo**: Presiona el globo para ver una animación de explosión con confeti

❤️ **Efectos Visuales**: Corazones animados, parallax y más

## 📁 Estructura del Proyecto

```
Para-el-mejor-pap-/
├── index.html      # Archivo HTML principal
├── styles.css      # Estilos y animaciones CSS
├── script.js       # Lógica interactiva con JavaScript
├── papa.jpg        # Imagen principal
└── README.md       # Este archivo
```

## 🚀 Cómo Usar

1. Abre el archivo `index.html` en tu navegador web
2. Desplázate por la página para ver todas las dedicatorias
3. Presiona el botón "¡Presiona el globo!" para ver la animación de explosión

## 🎯 Secciones

### 1. Imagen Principal
Una sección hero con la foto de la familia y un título elegante

### 2. Dedicatorias de Uriel
5 dedicatorias personalizadas con animaciones de entrada

### 3. Dedicatorias de Vitoria
5 dedicatorias personalizadas con un diseño alternativo

### 4. Globo Interactivo
Una animación de un globo que puedes presionar para ver confeti y un mensaje especial

## 📱 Responsive Design

La página se adapta automáticamente a:
- 📱 Teléfonos móviles (320px+)
- 📱 Tablets (768px+)
- 💻 Escritorio (1024px+)

## ✨ Animaciones Incluidas

- Fade In Up: Entrada suave de elementos
- Float: Efecto flotante en los personajes
- Balloon Float: Movimiento del globo
- Heart Beat: Latido de corazones
- Pop Message: Aparición del mensaje al reventar el globo
- Confetti: Explosión de confeti al presionar el globo

## 🔧 Tecnologías Utilizadas

- **HTML5**: Estructura semántica
- **CSS3**: Animaciones y diseño responsivo
- **JavaScript**: Interactividad y efectos dinámicos

## 💡 Personalización

Puedes personalizar fácilmente:
- Cambiar los colores en `styles.css`
- Modificar las dedicatorias en `index.html`
- Ajustar las animaciones en `script.js`
